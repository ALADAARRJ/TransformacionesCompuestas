REQUISITOS:
	1.- Tener instalado Visual Studio 2022, con la configuracion basica de instalacion
		para tener la posibilidad de abrir un programa en C o C++.
	2.- Tener algun programa para descomprimir archivos .ZIP

INSTRUCCIONES:

	1.- Descargar el .ZIP compelto directamente de GitHub.
	2.- Descomprimirlo en una carpeta donde todos los archivos puedan estar juntos.
	3.- Abrir Visual Studio 2022.
	4.- Abrir un nuevo proyecto y seleccionar la carpeta descomprimida.
	5.- Ya con el proyecto abierto, configurar la plataforma de solucion (alado de la opcion "Depurador local de Windows") en x86.
	6.- Compilar
	7.- Iniciar sin depurar.
	8.- Se abrirar dos nuevas ventanas, una en forma de terminal de windows, donde vendran las instrucicones para probar las funcionalidades del programa
	 y otra ventana .exe que debera estar en negro. Siguiendo las instrucciones de la terminal se podra interactuar con el programa.
	NOTA: RECUERDA QUE DEBERAS TENER ACTIVADAS LAS MAYUSCULAS PARA PODER USAR EL PROGRAMA.
